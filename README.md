# Lone Star Contacts

A demo contacts app built with [lone-star-ui](https://github.com/sethdavis512/lone-star-ui).

## Running locally

```bash
# Clone both repos side-by-side
git clone https://github.com/sethdavis512/lone-star-ui
git clone https://github.com/sethdavis512/lone-star-contacts

# Build the library
cd lone-star-ui && bun install && bun run build

# Run the app
cd ../lone-star-contacts && bun install && bun run dev
```
